\doxysubsection{Detailed Description}
Implementeaza notiunea abstracta de stare a jocului/programului. 

abstract class \mbox{\hyperlink{class_paoo_game_1_1_states_1_1_state}{State}} Un joc odata ce este lansat in executie nu trebuie \char`\"{}sa arunce jucatorul direct in lupta\char`\"{}, este nevoie de un meniu care sa contine optiuni\+: New \mbox{\hyperlink{class_paoo_game_1_1_game}{Game}}, Load \mbox{\hyperlink{class_paoo_game_1_1_game}{Game}}, Settings, About etc. Toate aceste optiuni nu sunt altceva decat stari ale programului (jocului) ce trebuiesc incarcate si afisate functie de starea curenta.
